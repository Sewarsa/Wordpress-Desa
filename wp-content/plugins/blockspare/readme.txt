== Installation ==
* Download to your plugin directory or simply install via WordPress admin interface.
* Activate.
* Go to page/post edit panel
* Click on "+" sign
* Get Blockspare blocks panel
* You will get beautiful bcustom blocks on WordPress editor to build your site.

== Frequently Asked Questions ==

**What are blocks?**

Blocks are the basic content building components, like buttons, images, videos, etc., for WordPress 5.0 / Gutenberg.

Blockspare provides you with lots of great blocks to create awesome posts, pages and front pages for you.

**Do I need to code?**

Nope, you do not need to have coding knowledge.

**Do I need to design the layout?**

No you don’t have to design the block because our plugin already has a range of pre-designed gorgeous layouts from which you can select one.

**Do I need to any third party page builder plugins?**

Definitely not, you do not need install any third party page builder plugin but you must have latest WordPress which supports block editor, namely Gutenberg.

**Can I use this with other block plugins?**

Oh yeah! With other blocks, Blockspare is performing well.

**Can I use Blockspare with any WordPress theme?**

Since Blockspare has its own designed components for website, it can be used for WordPress themes of any kind.

**Can I use Blockpare on client websites?**

Yes! You will definitely use the blocks of Blockpare on yours, as well as the websites of your client.

**Will Blockpare slow down my website?**

There's just no! The structure of the package and clean code keep it lightweight and simple. Blockpare blocks are designed with ease and performance in mind. Therefore, it can in no way impact the pace and efficiency of your website.

**Is Blockspare free?**

Yes, Blockspare is a free Building Block Plugin for WordPress Websites. But definitely, you can get more awesome controls on <a href="https://www.blockspare.com/pricing/" target="_blank" rel="">Blockspare Pro</a>

**Can I explore the block's demos?**

Definitely, you can explore all of the demos on <a href="https://blockspare.com/" target="_blank" rel="">Blockspare Website.</a>
